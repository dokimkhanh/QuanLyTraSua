
# Quản lý quán trà sữa

Simple milk tea shop management source code




## Features

- Manager Milk Tea
- Manager Table
- Manager Customer
- Manager Bill
- Export Bill PDF File
- Manager Statistical
- ...


## Tech Stack

**Client:** Entity Framework, Bunifu Framework

**Server:** SQL Server


## Environment Variables

To run this project, you will need create database name

`QuanLyMeoTea`
